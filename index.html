Hi, I am Zhiwei Jia...

You can also call me Sean. I am a sophomore at University of California, San Diego. I double majors in Computer Science and Applied Mathematics and have broad interest into these two areas. I study on web programming (both front-end and back-end), data mining, machine learning, computer vision, optimization and combinatorial algorithms. Currently I am building this website (updating contents and implementing responsive css) and converting some of the machine learning code from MATLAB to Python.

Personally I enjoy classical music very much. I have been playing the piano since I was five. Chopin's and Beethoven's works are my favorites. I have included a large amount of those classical pieces in this website. Click "Music" in the navigation bar to enjoy them freely.

This personal website is designed and implemented by myself. My selected personal works and collaborative projects are listed on the navigation bar. Scroll down for more details!

Overview of Selected Projects

  1. Finding Knight's Tour: A Java software for solving the ancient Knight's Tour puzzle. I designed and implemented this tool, with its multi-threading graph searching algorithm. It has a decent GUI with animation to show the solution. Click here to enjoy it online (I embedded JavaFx into HTML). Also, you can download it as a desktop App: for Mac OS X, or for Windows. Source code is available on github.
  2. Unit Calculator: A calculator that can understand unit. Specifically, users can define new units, simplify expressions with units, and do normal operations on numbers with units. I put this calculator to the server side (written in Java) so that users can use it on browsers. Enjoy it online! If you want to view the source code for this software, go to github.
  3. 2048: A special version of 2048 written in Java that the users can specify the size and shape of the board. I also built a GUI for it and it will be on this website soon. Welcome to view my source code and play the game on github.
  4. Nim: A classic combinatorial game. Easy rule and pretty interesting. I implemented the algorithm for this game and the code is write in Python 3.4 (NumPy package needed to run it). I am still testing its GUI. View it on github and try to beat the computer!
  5. Numerical Optimization Tools: I wrote code in MATLAB and built some tools to solve nonlinear equations, as well as for linear and nonlinear optimization. For linear programming, I used classic Simplex method in all-inequality form; for nonlinear programming, I use modified Newton method (with backtracking). Now you can download the code on github. Matlab compiler runtime evironment (MCR) needed.
